Admin

Sets custom user claims in Firebase Authentication.

Allows administrators to assign or update custom roles for a user.

Authorizations:
bearerAuth
Request Body schema: application/json
required
uid
required
string

claims
required
object

Responses

http://localhost:3000/api/nba/api/nba/admin/setUserClaims
